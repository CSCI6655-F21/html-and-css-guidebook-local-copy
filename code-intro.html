We're all familiar with websites in that we use them every day. On our phones, our desktops, our laptops, even our watches. But to be able to make a website, you need to really understand what exactly a website is on a fundamental level, beyond what a casual user sees.

What is the Internet?

The internet is basically a network of computers; that is, a bunch of computers that are able to talk to one another. This includes all kinds of computers, but most importantly for our purposes, it includes a special type of computer called a server.

A server is like a house for web content.

The purpose of a server is to store and provide access to all sorts of web content, from video files, to images, to databases of information. And most fundamentally, servers store the code files from which websites are made.

We'll talk more about code files shortly.

IP Address

Every computer on the internet has a unique address called an IP (Internet Protocal) Address. This includes all the devices you use to access the internet: your phone, your PC, even smart lightbulbs. The IP address is how computers find each other to have a conversation. And of course this includes servers.

So when your computer wants to talk to a server that houses the content for a particular wenbsite, it has to find that server using its unique IP address.

The problem is that IP addresses are long strings of numbers and dots (e.g. 123.456.78.90), which aren't very easy for humans to remember. It would be terrible if I had to manually enter a number like that for every site I wanted to visit!

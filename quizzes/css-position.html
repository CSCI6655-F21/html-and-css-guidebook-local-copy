Moving in the Viewport

And now something a little more complex and position an element so that it stays in place as the page scrolls beneath it. Again, we'll do this in two steps:

  1. Set the position property for the navigation so that it is removed from the document flow and can be positioned with regard to the viewport window as a canvas. It should remain still when the page scrolls.
  2. Place the navigation at the very bottom of the viewport window, with no space between it and the edge of the window.

Hint: Make sure your properties are in alphabetical order.
